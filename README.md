# Расчет потребности в кормах

Проект выполнен для варианта 1. Приложение на PyQt5 рассчитывает потребность в кормах и строит точечный график сравнения с данными за прошлый и позапрошлый год.

## Запуск

Самый простой способ на Windows:

```powershell
run.bat
```

Файл `run.bat` сам создаст виртуальное окружение `.venv`, установит PyQt5 из `requirements.txt` и запустит программу.

Ручной запуск:

```powershell
pip install -r requirements.txt
python main.py
```

Если в PyCharm создан отдельный интерпретатор, установите зависимости в его Terminal.

## Как передать проект другому пользователю

Передавайте архив проекта без папок `.venv`, `.git`, `.idea` и `__pycache__`. Эти папки создаются локально и содержат пути конкретного компьютера.

В архиве должны быть:

- `main.py`
- `requirements.txt`
- `pyproject.toml`
- `README.md`
- `run.bat`
- `run.ps1`

На другом компьютере нужно распаковать архив и запустить `run.bat`. Требуется только установленный Python с добавлением в PATH.

## Что есть в приложении

- поле ввода поголовья коров;
- поле ввода количества дней содержания;
- выбор вида корма: концентраты, сено, силос;
- кнопка расчета потребности;
- отдельное поле результата;
- точечный график сравнения с 51000 ц и 29500 ц за прошлые периоды.

Формула:

```text
Потребность = поголовье коров * количество дней содержания * норматив потребности на 1 голову
```

Нормативы:

| Вид корма | Потребность на 1 голову в день |
| --- | ---: |
| Концентраты | 5 |
| Сено | 15 |
| Силос | 20 |
